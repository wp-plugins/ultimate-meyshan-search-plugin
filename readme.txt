This plug in is used to search your blog for comments, images, post, achieves and everything. Live search style. 

== Description ==

* User can search blog pages that is listed in Yahoo and Google sites.
* Meyshan King search widget integrated with Google custom search,Yahoo web search,video search, blog search, and local search.
* This widget is easy to use, since it is in-built with snap preview which allows us to see the preview before clicking on the link.
* This widget displays the search results in a new window when clicked using yahoo exit, keeping the current page as such.
* It has Google Adsense integrated, so that it allows you to get some money!
    

== Installation ==

This section describes how to install the plugin and get it working.

1.Open the link
http://www.spicyexpress.net/general/ultimate-meyshan-king-search-wordpress-widget/

2.Download the zip file to your local system by clicking the download link.

3.Unzip the file and copy the folder 'Meyshan search king' to the plugin folder into your plugin directory.

4.Now active the plugin,it will automatically over write your default search. 

5.Through admin go to option page.

6.Since this widget needs some API keys,Get Google api key, adsense code(optional) custom search link (optional).

7.Configure these keys under option in your wordpress admin. 

8.Once this is configured completely,You can control where to display the Google adsense, either on the top or in the right side. 

  Thats it!!
